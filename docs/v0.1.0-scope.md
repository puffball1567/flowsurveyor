# v0.1.0 Scope

FlowSurveyor v0.1.0 is a complete first release for offline flow analysis.

## Included

- In-memory graph and event model
- Node duration, success, and failure aggregation
- Edge duration, success, and failure aggregation
- Critical path analysis for directed acyclic graphs
- Bottleneck ranking by duration and failure impact
- Rule-based recommendations
- Event quality checks
- Variant duration/failure comparison
- JSON report export with `schemaVersion = 1`
- Integration-safe `analyze` API with `ok`, `report`, and `errors`
- Unit tests, example, benchmark, design notes, and IP notes

## Not Included

- Task execution
- Scheduling
- Event persistence
- External telemetry collection
- Dashboard rendering
- Automatic graph mutation or optimization
- Database, queue, web framework, or runner-specific adapters

## Completion Checklist

- [x] Public API exports match the documented v0.1.0 scope

- [x] Recommendations are documented as rule-based hints
- [x] FlowCaptain-style callers can use `analyze` without exception-based

- [x] Example and benchmark are present
